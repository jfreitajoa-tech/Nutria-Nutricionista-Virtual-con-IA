# Nutria – Nutricionista virtual con IA

Nutria es un proyecto web cuyo objetivo es ofrecer una ayuda básica en nutrición de forma sencilla, cercana e interactiva. La idea principal es que el usuario pueda mejorar sus hábitos alimenticios con el apoyo de una interfaz amigable y una mascota que guía la experiencia.

## Descripción

El proyecto consiste en una página web donde se presenta el concepto de Nutria como asistente nutricional. A través de la mascota animada y un chat integrado, el usuario puede interactuar y recibir respuestas relacionadas con alimentación, hábitos y objetivos personales.

Se ha buscado que la experiencia sea clara, visual y fácil de usar desde el primer momento.

## Funcionalidades

- Animación inicial de la mascota mediante SVG
- Aparición progresiva de la nutria desde una silueta
- Interacción con la mascota (movimiento, reacción)
- Chat básico con respuestas según palabras clave
- Interfaz responsive adaptada a distintos dispositivos
- Diseño visual enfocado en una experiencia cercana

## Tecnologías utilizadas

- HTML
- CSS (Tailwind)
- JavaScript
- SVG para animaciones

## Objetivo del proyecto

Este proyecto ha sido desarrollado como parte del ciclo formativo de Desarrollo de Aplicaciones Web (DAW). El objetivo principal es aplicar conocimientos de desarrollo front-end, mejorar la experiencia de usuario y crear una interfaz interactiva con identidad propia.

## Estructura

- index.html → estructura de la web
- script.js → lógica e interacciones
- assets/ → imágenes, vídeos y recursos

## Ejecución

No requiere instalación. Solo es necesario abrir el archivo index.html en el navegador o acceder a la versión publicada en GitHub Pages.
